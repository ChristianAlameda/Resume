  • Website Status Checker

    This program, developed for California State University Stanislaus' CS-4800 2023 Fall class with Dr. Hatem, monitors server health by pinging servers to check for responsiveness and latency. It allows users to set a polling rate and records server status and latency in a database for later visualization through histograms. The program is accessible on a local machine or the web (subject to firewall permissions) and supports pinging devices on both Local Area Networks (LAN) and Wide Area Networks (WAN).
